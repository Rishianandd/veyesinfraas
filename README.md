## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Rishianandd/veyesinfraas.git
```

Navigate to the project:

```bash
cd veyesinfraas
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
